# Netflix Movies and TV Shows - Exploratory Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

## Dataset Source

Kaggle: Netflix Movies and TV Shows Dataset

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

* Dataset Overview
* Missing Values Analysis
* Duplicate Rows Check
* Movies vs TV Shows Analysis
* Top 10 Countries Analysis
* Top 10 Genres Analysis
* Ratings Distribution
* Content Growth by Release Year
* Content Added by Year

* dataset/
* reports/
* visuals/
* all_eda_code.py
